# AI Usage Log – Milestone 1

|AI Tool Used | Prompt Submitted | Purpose of Prompt | How Results Influenced Design |
|-------------|------------------|------------------|------------------------------|
| ChatGPT | Give me software project ideas for college students using full stack development. | Brainstorm project concepts. | Helped narrow ideas to a creative content-sharing platform because it matched team interests and skills. |
| ChatGPT | What features should a platform for writers, musicians, and artists include? | Explore possible MVP features. | Generated ideas like profiles, uploads, comments, and categories which became core system features. |
| ChatGPT | Create user stories for a creative social media platform. | Help define functional requirements. | User stories were adapted into initial backlog items and project scope. |
| ChatGPT | What technologies are best for a MERN stack student project? | Select technology stack. | Confirmed use of MongoDB, Express, React, and Node.js for manageable full-stack development. |
| ChatGPT | What are common risks for student software projects? | Identify project risks. | Led team to focus on smaller MVP scope and divide tasks early. |
